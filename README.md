checker
=======


Checker is a simple hello world firefox plugin.
You could find the same on firefow home pages.

How to build ?
--------------


```shell

cd $CHECKER_DIR `
cd src
make clean
make

```


How to run ?
------------

Go to the bin directory :

```shell
cd $CHECKER_DIR/bin
```
And drag and drop on firefox.
It will ask you to install it.
